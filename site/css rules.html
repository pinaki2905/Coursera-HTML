How Can You Code Cascading Style Sheet Rules?

CSS rules is presentation layer. It helps browser present the HTML content in beautiful manner. There're numbers of selectors, namely...

  • Element Selector
  • Class Selector
  • Id Selector

Element Selector

Element Selectors modify all the content with target html element. Foe example, p {}- is a element selector and will modify all the content within <p> html element.

Class Selector

This selector will modify all the content within the class having perticular attribute as specified in the css rules. For example, ..heading {}- is a class selector which affect all the content within the class= heading

Id Selector

This selector will affect the content which is within that perticular id element as specified in css rule. Let's take an example... #main will affect the conent only within.... <span id="main">

As you can see that id selector is most specific amongst all as there'll be only one id associated with the entire content.
